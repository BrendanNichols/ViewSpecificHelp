This app works by checking what is relevent in your Revit Application

If you have elements selected when pressing the button, you will be guided to the help page for the first item selected.

Try selecting only one thing.

If you have no elements selected, you will be guided to the help page for the current view template or view name.

If you've created your own view template or used naming conventions different from the project template you won't find the help you need. If you've found a bug or would like more content for you're current training needs please contact Brendan Nichols
